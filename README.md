# Social Media Sentiment Analysis

This project performs sentiment analysis on a dataset of **1.6 million tweets** using Python and Natural Language Processing (NLP) techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLP

## Project Workflow

1. Data cleaning and preprocessing
2. Text vectorization
3. Model training
4. Sentiment classification (Positive, Negative, Neutral)

## Result

The model analyzes tweets and predicts sentiment to understand public opinion trends.
